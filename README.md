# Modiva — Premium Fashion E-Commerce 👗✨

![Modiva Banner](public/images/readme/mockup.png)

Modiva is a high-end, minimalist e-commerce storefront designed for a professional fashion brand experience. Built with a focus on visual excellence, responsive design, and robust cloud integration.

##  Live Demo
Experience the live site here: **[https://modiva-store.web.app](https://modiva-store.web.app)**

##  Project Showcase
The storefront features a custom "Product-Only" aesthetic, using professionally generated flat-lay and ghost-mannequin photography to emphasize a luxury brand identity.

![Homepage Screenshot](public/images/readme/homepage.png)

## Key Features
- **Modern UI/UX**: minimalist aesthetic with glassmorphism, gold/emerald color palette, and smooth micro-animations.
- **Firebase Authentication**: Secure user login and registration flow.
- **Firestore Database**: Real-time product synchronization and inventory management.
- **Dynamic Shopping Cart**: Fully functional cart with off-canvas previews and persistent state.
- **Product-Only Aesthetic**: Professional fashion photography assets (No models) for a clean, boutique feel.
- **Admin Dashboard**: backend interface for product management (CRUD operations).
- **Responsive Design**: Fully optimized for mobile, tablet, and desktop views.

##  Tech Stack
- **Frontend**: HTML5, CSS3 (Vanilla), Bootstrap 5.3
- **Backend/Cloud**: Firebase (Authentication, Firestore, Hosting)
- **Logic**: Vanilla JavaScript (ES6+)
- **Assets**: Custom AI-generated high-fidelity fashion photography.

##  Local Setup
To run this project locally:

1. **Clone the repository**:
   ```bash
   git clone https://github.com/rahma-khaled/Clothing_E-Commerce.git
   ```
2. **Install Firebase CLI** (if you haven't):
   ```bash
   npm install -g firebase-tools
   ```
3. **Run a local server**:
   You can use `npx serve public` or any static server to view the `public` directory.
   ```bash
   npx serve public
   ```
4. **Deploy**:
   ```bash
   firebase deploy
   ```
